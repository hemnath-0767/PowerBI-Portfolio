# Brazil Sales Dashboard | Power BI Portfolio Project

> An interactive Power BI dashboard built to analyze sales performance across Brazil using data visualization, DAX, Power Query, and business intelligence techniques.

---

## 📖 Overview

The **Brazil Sales Dashboard** provides a comprehensive view of business performance by transforming raw sales data into meaningful insights. It enables users to monitor key performance indicators, analyze sales trends, identify high-performing products and regions, and make data-driven business decisions.

This project demonstrates the complete Power BI workflow—from data cleaning and transformation to data modeling, DAX calculations, and interactive dashboard design.

---

## 🎯 Business Objectives

- Monitor overall sales performance
- Analyze revenue and profit trends
- Identify top-performing products and categories
- Compare regional sales performance
- Understand customer purchasing behavior
- Build an interactive dashboard for business users

---

- Sales Trend Over Time

### Product Analysis
- Top Selling Products
