# Brazil Sales Dashboard | Power BI

## 📌 Project Overview

This Power BI dashboard analyzes sales performance across Brazil, helping users understand business trends through interactive visualizations and KPIs.

The dashboard enables users to monitor sales, profit, customer performance, product categories, and regional insights for better business decision-making.

---

## 🎯 Objectives

- Analyze overall sales performance
- Monitor profit and revenue trends
- Identify top-performing products and categories
- Compare sales across regions
- Track customer purchasing behavior
- Create an interactive and user-friendly dashboard

---

## 📊 Dashboard Features

- KPI Cards
  - Total Sales
  - Total Profit
  - Total Orders

- Sales Trend Analysis

- Product Category Analysis

- Regional Sales Analysis

- Customer Insights

- Interactive Filters (Slicers)

---

## 🛠 Tools Used

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel / CSV Dataset
